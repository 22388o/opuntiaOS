## Features
### Kernel
* x86-32 and ARMv7 kernel with pre-emptive multi-threading
* Ext2 FS
* Local sockets
* /dev filesystem
* POSIX signals
* TTY

### Libraries
* Custom LibC (in development)
* Custom LibCXX
* LibFoundation
* LibG & LibUI

### Userland
* Basic userland
* Edit - a simple clone of vim
* Composing windows manager
* Simple gui-app "About"
* Terminal
